OpenID Federation for JavaScript provides a standards-compliant specification implementation, including production-grade library code, live playground ([explore.oidfed.com](https://explore.oidfed.com/)), central documentation ([oidfed.com](https://oidfed.com/)), and comprehensive learning resources ([learn.oidfed.com](https://learn.oidfed.com/)). This project enables developers and organizations to adopt and integrate OpenID Federation protocols in JavaScript applications with reliability and interoperability.

Other public work includes Parako.ID, a fully self-hosted and free OpenID Connect server ([parako.id](https://parako.id/), [docs.parako.id](https://docs.parako.id/)); the community-driven documentation portal for @panva/node-oidc-provider ([oidc-provider.dev](https://oidc-provider.dev/)); [fillkit/sdk](https://github.com/fillkit/sdk) and [fillkit.dev](https://fillkit.dev/), an open-source form filling toolkit for development and test; Benin location and administrative data as a ready-to-use JavaScript package ([location_data_bj_js](https://github.com/Dahkenangnon/location_data_bj_js)); and OHADA region accounting integration for JavaScript ([ohadakit](https://github.com/Dahkenangnon/ohadakit)). All projects are available for direct adoption, contribution, and integration.
